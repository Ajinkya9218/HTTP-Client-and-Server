
Instructions to run the project:

1. Server executing commands:

Type the following command on command prompt to compile the java file "Serverside.java"

javac Serverside.java

Type the following command on command prompt to run the server

java Serverside <port-number>
e.g java Serverside 7059


2. Client executing commands:

Type the following command on command prompt to compile the java file "Clientside.java"

javac Clientside.java

Type the following command on command prompt to run the client
java Clientside <hostname> <port-no> <GET/PUT> <filename>

e.g java Clientside localhost 7059 GET CCN.html



Note: Use port nos. greater than 5000.
      Type ‘STOP’ to interrupt the server and allow it to stop it.
